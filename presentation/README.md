# Medisync Presentation

This project is a web-based presentation for **Medisync**, an AI-Powered Hospital Orchestration System, built using [Reveal.js](https://revealjs.com/).

## Overview

The presentation outlines the Medisync proposal, covering:
- **Introduction:** Background on hospital inefficiencies.
- **Problems:** Coordination, workload balance, and execution delays.
- **Objectives:** Staff utilization, timely execution, and error reduction.
- **Solution:** A centralized AI orchestrator (Medisync).
- **Impact:** Social, Economic, and Organizational benefits.

## How to Run Locally

Since this is a static HTML site using Reveal.js via CDN, you can simply open the file in your browser.

1.  Clone this repository.
2.  Open `index.html` in your preferred web browser (Chrome, Firefox, Safari, etc.).

## Deployment

This project is configured to automatically deploy to **GitHub Pages**.
Any push to the `main` branch will trigger the deployment workflow.
